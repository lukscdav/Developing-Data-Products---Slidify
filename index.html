What is BMI

  • The Body Mass Index (BMI) is a measure of relative weight based on an individual's mass and height.
  • Formula: \(BMI=\frac{mass(lb)}{height(in)^2}\times{}703\)
  • Formula (metric units): \(BMI=\frac{mass(kg)}{height(m)^2}\)
  • BMI < 18.5: underweight
  • 18.5 < BMI < 25: normal weight
  • BMI > 25: overweight

Examples

  • Example (using imperial units): 
height <- 69
weight <- 187
print(BMI <- weight/height^2*703)

## [1] 27.61
  • Example (using metric units): 
height <- 1.76
weight <- 85
print(BMI <- weight/height^2)

## [1] 27.44
